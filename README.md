# Product Recommendation System

This project is a Machine Learning based Product Recommendation System developed using Python and Streamlit.

## Features
- Product recommendations using clustering
- KMeans, Hierarchical & DBSCAN comparison
- Rating and popularity filters
- Interactive Streamlit UI

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib
- Seaborn

## Run Project

streamlit run app.py
